#  Tastebud Traveller

A responsive food delivery web application built using HTML, CSS, JavaScript, and Bootstrap.

---

## 🚀 Features

* Dynamic food menu with filtering
* Shopping cart with real-time price calculation
* Persistent cart using localStorage
* Multi-step checkout flow (Address & Payment simulation)
* Responsive design for mobile and desktop

---

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript
* Bootstrap

---
